# ระบบพลัง (Power System)

ระบบพลังคือหัวใจของเนื้อเรื่อง เพราะการเติบโตของนูระ = การปลดล็อกพลังทีละขั้น

## 1. เวทย์แสงศักดิ์สิทธิ์ (Holy Magic)
- สายของแม่ **ริวเซน ฮิเมะ** — การรักษา, การชำระล้าง, แสงป้องกัน
- บริสุทธิ์ ให้ชีวิต แต่ถ้าใช้เพื่อทำลายจะทรงพลังมหาศาล
- นูระ **ยังใช้ไม่ได้** ในตอนแรก (หลับใหลอยู่ในสายเลือด)

## 2. เวทย์มืด (Demon Magic)
- สายของพ่อ **อารากอน** — พลังทำลายล้าง, การครอบงำ, เงา
- อันตราย เสี่ยงกลืนกินสติของผู้ใช้ถ้าคุมไม่อยู่
- นูระ **ยังใช้ไม่ได้** เต็มรูปแบบเช่นกัน

## 3. เนตรมาร (Demon Eye) — พลังเดียวที่นูระใช้ได้ตั้งแต่ต้น
- มองทะลุการเคลื่อนไหว, อ่านเจตนา, เห็นการไหลของพลังเวทย์
- **ยังไม่สมบูรณ์** ในตอนต้น — ติด ๆ ดับ ๆ เวลาคับขัน
- พัฒนา 3 ขั้น: กระพริบ (S1 ต้น) → ตื่นครั้งแรก (S1 จบ) → สมบูรณ์ (S2)
- คำพ่อ: "อย่าให้ใครมองทะลุผ่านเนตรมารของเจ้า"

## 4. พลังกายภาพ & ดาบคานาตะ (Physical + Kanata Blade)
- จุดแข็งที่แท้จริงของนูระตอนนี้ — ฝีมือดาบระดับเทพ + พลังกายเหนือมนุษย์
- ฝึกจากการประลองจริงกับ **มหามารทั้งสี่** และ **อัศวินศักดิ์สิทธิ์ทั้งสี่** ทุกวัน
- อายุ 18 สามารถสู้ 1 v 8 (มหามาร 4 + อัศวิน 4) ได้ด้วยดาบเล่มเดียว
- พลังฟื้นฟูขั้นต่ำ (regen) — บาดเจ็บหายเร็วกว่ามนุษย์ทั่วไป

## 5. เลือดสองสาย (The Two-Blood / Dual-Blood) — พลังสูงสุด
- การหลอมรวมแสง + มืด อย่างสมดุลสมบูรณ์ในร่างเดียว
- เหนือกว่าพลังที่ **เรย์จิ** ดูดกลืนจากจอมมารเมื่อ 300 ปีก่อน
- คำแม่ (คำพยากรณ์หลักของเรื่อง):
  > "เมื่อดาบของลูกไม่พออีกต่อไป… พลังที่แท้จริงในตัวลูกจะตื่นขึ้นเอง"
- **ตื่นเต็มรูปแบบใน Season 2** — คือ climax ของทั้งเรื่อง
