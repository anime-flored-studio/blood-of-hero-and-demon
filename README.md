# Blood of Hero and Demon
### บุตรแห่งแสงและความมืด (Son of Light and Darkness)

โปรเจกต์อนิเมะ / ไลท์โนเวล — เรื่องราวของ **นูระ** บุตรชายเพียงคนเดียวของ **ผู้กล้าแสงบริสุทธิ์** กับ **จอมมารแห่งความมืด** ผู้ถูกเทพจากอีกโลกเชิญให้ข้ามมิติไปกอบกู้โลกเอลเดเรีย

- **Season 1** — 13 ตอน — ปฐมบท: การมาถึง และความจริงเบื้องหลัง "ผู้กล้า"
- **Season 2** — 14 ตอน — สงครามแห่งแสงและความมืด และการตื่นของเลือดสองสาย

## เริ่มอ่านจากไหนดี
1. `docs/STORY.md` — เนื้อเรื่องภาพรวม
2. `docs/power-system.md` — เข้าใจระบบพลัง (สำคัญมากต่อเนื้อเรื่อง)
3. `seasons/season-01/season-01-overview.md` — แผนตอน Season 1
4. ไล่อ่านตอนละไฟล์ใน `seasons/*/episodes/`
