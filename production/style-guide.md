# Style Guide (Production) — โครงร่างสำหรับอนาคต

> เตรียมไว้เติมเมื่อเข้าสู่ขั้นผลิตจริง

## ทิศทางภาพ (Visual Direction)
- โทน: dark fantasy + isekai — เริ่มสว่างอบอุ่น ค่อย ๆ มืดลง
- Key visual cue: **เนตรมาร** (ม่านตาเปลี่ยนสี/ลาย), **ผมเงิน** ของนูระ, ดาบคานาตะ

## พาเลตต์สี (Color Palette)
- แสง (แม่/ริวเซน): ทอง-ขาว-ฟ้าอ่อน
- มืด (พ่อ/อารากอน): ม่วงเข้ม-ดำ-แดงเลือด
- เลือดสองสาย (นูระตื่นรู้): หลอมทองกับม่วง/แดง เป็นสัญลักษณ์สมดุล

## ดีไซน์ตัวละคร
- อ้างอิงภาพใน `characters/art/`
